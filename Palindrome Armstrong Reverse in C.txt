WAP to take a number from a user and print whether it is palindrome or not.

Palindrome logic:

If a number and its reverse is same, then it is called palindrome.

1. User input
2. Reverse
3. If reverse and input number is same, it is palindrome or not.


#include<stdio.h>
#include<conio.h>
void main()
{
	int num=0,rev=0,rem=0,temp=0;
	clrscr();
	printf("Enter a number:");
	scanf("%d",&num);
	temp=num;
	while(num>0)
	{
   		rem=num%10
		rev=rev*10+rem//reverse formula.
		num=num/10
	}
	if(temp==rev)
	{
		printf("\nThe given number is palindrome");
	}
	else
	{
		printf("\n The give number is not palindrome");
	}

getch();

}

WAP to take a number from a user and check whether it is Armstrong number or not.

Armstrong number:
An armstrong number is that number when the sum of individual powered digits by the total count ko no of digits is equal to the original number.

3 digit:A number is an armstrong number if the original number and sum of cubes of individual digit of that number is same.
4 digit: to the power 4.
153=1 cube + 5 cube +3 cube
1234= 1 to the power 4+2 power 4+3 power 4+5 power 4

1634:
number:234:8+27+64:99
       153:1+125+27:153, hence 153 is armstrong number

		rem=temp%10;  //6    //1     //5     //1
		count=count+1; //1   //2      //3     //4
		temp=temp/10;  //151   //15   //1      //0

	}
	while(num>0)
	{
   		rem=num%10;
