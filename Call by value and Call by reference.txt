Call by value and Call by reference:


Call by value:

While calling a function, we pass values of variables to it. Such functions are known as “Call By Values”.

In this method, the value of each variable in calling function is copied into corresponding dummy variables of the called function.

With this method, the changes made to the dummy variables in the called function have no effect on the values of actual variables in the calling function.
eg: call by value:

Eg no 1:
#include<Stdio.h>

int sum(int,int);

void main()
{

	int a,b,result=0;
	clrscr();
	printf("Enter 2 numbers a and b");
	scanf("%d %d",&a,&b); //a=5,b=6
	result=sum(5,6); //function call
	/* Here while we are doing function call, we are sending value of variables a nd b. Thats why this program is example of call by value).
	printf("\n The result is %d",result); //11
	getch();	
}
	

int sum(int x,int y)
{
	int z;
	z=x+y; //11
	return z;//11
}


Eg no 2:

#include<stdio.h>

// Function Prototype
void swapx(int x, int y);

// Main function
int main()
{
    int a = 10, b = 20;

    // Pass by Values
    swapx(a, b); (10,20)

    printf("a=%d b=%d\n", a, b);

    return 0;
}

// Swap functions that swaps
// two values
void swapx(int x, int y) (x=10,y=20)
{
    int t;

    t = x;
    x = y;
    y = t;

    printf("x=%d y=%d\n", x, y); (20,10)
}

Output:
x=20 y=10
a=10 b=20


Call by reference:

While calling a function, instead of passing the values of variables, we pass address of variables(location of variables) to the function known as “Call By References."
Pointer variables are necessary to define to store the address values of variables.With this method, using addresses we would have an access to the actual variables and hence we would be able to manipulate them.

eg: 1

#include<Stdio.h>

int sum(int *,int *);

void main()
{

	int a,b,result=0;
	clrscr();
	printf("Enter 2 numbers a and b");
	scanf("%d %d",&a,&b);//
	result=sum(&a,&b); //function call
	/* Here while we are doing function call, we are sending value of variables a nd b. Thats why this program is example of call by value).
	printf("\n The result is %d",result);
	getch();	
}
	

int sum(int *x,int *y)
{
	int z;
	z=*x+*y;
	return z;
}


eg 2:

#include <stdio.h>

// Function Prototype
void swapx(float *, float *);

// Main function
int main()
{
    float a = 10, b = 20;

    // Pass reference
    swapx(&a, &b); // address of a and b is passed in function call. value of a=10, value of b=20.

    printf("a=%f b=%f\n", a, b);

    return 0;
}

// Function to swap two variables
// by references
void swapx(float * x, float * y) // 
{
    float t;

    t = *x;
    *x = *y;
    *y = t;

    printf("x=%f y=%f\n", *x, *y);  

}

output:
x=20,y=10.
a=20,b=10.
