How to pass a function as a parameter inside an array?

Ans:

In case of passing variable as a parameter:

a. Function declaration:
  let us suppose i am passing 1 integer variable in a function.

Syntax:
    <return type> function name(int);

let us suppose i am passing 3 integer variable in a function.

Syntax:

  <return type> function name(int,int,int);

let us suppose i am passing 1 integer,1 decimal and 1 character variable in a function.

Syntax:

<return type> function name(int,float,char);


In case of passing array as a paramter:

float a[10];=a is an array //float a= a is variable
int c[10];
char x[10];


In declaration section:
If you are passing a[10] array,which is of int data type.

Syntax:
<return type> function name (int x[]);= If you do this, then compiler will understand,
that the function accepts only array as a parameter.

But

<return type> function name (int);= If you do this, then compiler will understand,
that the function accepts only a single variable which is of integer data type as a parameter.



In case of array, function call:

Rule: Inside the paranthesis, we write only array name like we used to write variable name while passing variable.



Example: Passing array in a function:





Things to know:
//float avg(float); = In this we are passing, a float variable.
//float avg(float x);= This is same thing as above.

float avg(float x[]);  // In this we are passing an array as function parameter, and returning a float variable.

If we want to pass array in function and also return array,Then in this case, we should do:

float * avg(float x[]); (We will study this portion in pointer chapter).

		printf("\n Enter %d number",i+1);
		scanf("%f",&a[i]);

float avg(float m[])  //This is function definition.
{

	float average=0,s=0;
	int j=0;

		s=s+m[j];
	}
	average=s/10;
	return average;

int length(char m[])
{
	int i=0,count=0;
	for(i=0;m[i]!='\0';i++)
	{
		count++;
	}
	return count;
}
