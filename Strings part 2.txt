3. WAP to print sentence "We are in lockdown."

#
#
void main()
{
    char a[]="We are in lockdown."; or char a[20]="We are in lockdown.";
    printf("The entered sentence is %s",a);
    getch();
}

Note: Dont use %s ever while doing program related to sentence. %s is only ok
 if you are dealing with single word.
 The above program will have following output:

Output:
We (%s wont deal after space).

So while dealing with sentence we always use puts to display.
So correct program is


#
#
void main()
{
    char a[]="We are in lockdown."; or char a[20]="We are in lockdown.";
    printf("The entered sentence is:");
    puts(a);
    getch();
}
Output:
The entered sentence is: We are in lockdown.

ASCII values:

ASCII: American Standard Code For Information Interchange

Each and every number letter and symbols have their respective ASCII value.

a-z:97-122
A-Z:65-90


#
#
void main()
{
    char a;
    clrscr();
    printf("Enter a character:);
    scanf("%c",&a);  //m
    printf("The ASCII value of %c is %d",a,a);
    getch();
}

Output:( If you display character using %c, then it displays that character, But if you display same character using %d, instead of display character it will display its
ASCII value)
Enter a character:m
The ASCII value of m is 109.
