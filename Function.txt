What is function?
 
Function is a block of code that performs a specific task and can be used repeatedly again and again whenever necessary.

strlen(): It is a function.

Why do we need function?

Letâ€™s say you are writing a C program and you need to perform a same task in that program more than once. In such case you have two options:

a) Use the same set of statements every time you want to perform the task.
b) Create a function to perform that task, and just call it every time you need to perform that task.


Types of function:

There are two types of function in C programming:

1. Standard library functions: 
Functions which are already defined in C by the system itself with a library file.

For example: If we want to use printf(), we need to write include<stdio.h>. Here printf is a function which perform a task by printing whatever user has typed in between the paranthesis (""). And stdio.h is a library file, where the action related to printf is already defined.

So whenever compiler meets printf(), it checks upon  the stdio.h library file and perform the action.

Another example: strrev(): Strrev() is a string handling function which reverses the string. The reversing code is already listed in the library file, string.h. So whenever the compiler meets strrev(), it checks its operation from string.h library file and perform the related task.

But what if? We could write the code only once, since the nature for all case is same

So function is a better approach, if we are dealing with same block of code over and over again. For which the programmer has to build a function himself. 
In above example, programmer has to build a function himself/herself in order to solve
the task. Such functions created by user in order to solve a specific task is known as user-defined function.
